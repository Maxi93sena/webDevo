Step 2: Basic HTML Structure

HTML forms the structure and content of your website. It uses tags to define elements like headings, paragraphs, links, images, and more.

Here are some common HTML tags:

  • <h1> </h1> - Main heading
  • <p> </p> - Paragraph text
  • <a href="url"> </a> - Link to another page or website
  • <img src="image.jpg" alt="description"> - Image
  • <ul> <li> </li> </ul> - Unordered list and list items

Example of a simple webpage with these elements:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Website</title>
</head>
<body>
  <h1>Welcome to My First Website</h1>
  <p>This is a paragraph introducing my site.</p>

  <h2>My Hobbies</h2>
  <ul>
    <li>Drawing</li>
    <li>Playing guitar</li>
    <li>Coding</li>
  </ul>

  <p>Check out my <a href="https://www.example.com" target="_blank">favorite website</a>!</p>

  <img src="https://via.placeholder.com/300" alt="Example Image">
</body>
</html>

This shows how to build the page with headings, paragraphs, lists, links, and images.

Step 3: Introduction to CSS

CSS (Cascading Style Sheets) is used to style your HTML elements — colors, fonts, spacing, layout, and more.

You can add CSS inside your HTML using the <style> tag or in a separate file.

Basic CSS Syntax:

selector {
  property: value;
}

Example: Changing the background color and text styles

body {
  background-color: #f0f8ff;
  font-family: Arial, sans-serif;
  color: #333;
}

h1 {
  color: #4CAF50;
  text-align: center;
}

p {
  font-size: 16px;
  line-height: 1.5;
  margin: 10px 0;
}

Example explained:

  • body selector targets the whole page background and font.
  • h1 targets all main headings and colors them green and centers text.
  • p targets all paragraphs, sets font size and spacing.

You can apply CSS rules to make your website look beautiful and unique!

Step 5: Publishing Your Website

After creating your website, the final step is to publish it so others can see it online. Here's a detailed guide for publishing your website easily:

Using GitHub Pages (Free & Popular for Beginners)

  1. Create a GitHub Account: Go to GitHub and sign up for a free account if you don't have one.
  2. Create a New Repository: After logging in, click the "+" icon at the top-right corner and select "New repository". Name it something like my-website.
  3. Upload Your Files: In your new repository, click "Add file" → "Upload files". Upload your website files including the HTML file you created with the template generator. Commit the changes.
  4. Enable GitHub Pages: Go to the "Settings" tab of your repository, scroll down to the "Pages" section.
  5. Select Source: Under "Branch", select main (or master if your repository uses that branch) and click "Save".
  6. Access Your Website: GitHub will display your website URL (usually https://yourusername.github.io/repositoryname). It might take a few minutes for your site to go live.

Tips:

  • For your homepage, name the main file index.html; GitHub Pages automatically loads it.
  • If you want to update your website later, upload new files or edit existing ones in your repository.
  • GitHub also lets you edit files directly in the browser for quick changes.

Other Easy Platforms

If you find GitHub Pages a bit challenging, here are other platforms with user-friendly interfaces:

  • Netlify - Drag and drop your site folder to deploy instantly.
  • Vercel - Simple deployments with free hosting.
  • Glitch - Online editor and hosting for small projects.

Publishing your website is easier than you think! Take your time, follow these steps, and soon your site will be live for the world to visit.
